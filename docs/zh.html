⌘ QuickAdd

随手记下，
一个快捷键的距离。

免费、开源的 macOS 菜单栏 App。在任何界面按 ⇧⌘A，像说话一样打字， 它就把提醒或日历事件建好——时间、地点、优先级、重复都自动填好。

下载 macOS 版 在 GitHub 查看

自然地打字，剩下的它来搞定

有时间段或时长 → 事件；有地点或会议关键词 → 事件；其余都是提醒。

明天下午3点 开会 30min→ 日历事件，15:00–15:30
周五 9点-10点 团队会议→ 带时间段的事件
客户会议 明天下午3点 建国路88号→ 事件，地址 → 地点 📍
买菜 ~杂货 #周末 !!→ 高优先级提醒，带清单和标签
每周一 上午10点 周会→ 每周重复的提醒

功能

⌨️ 全局快捷键

Spotlight 式面板浮在任何 App 之上，用完自动把焦点还给你。

🧠 自然语言

时间、时长、区间、重复、优先级——从你打字的方式里直接解析出来。

📍 智能判别

知道"在某地址开会"是事件，并自动把地址提取成地点。

🔎 统一搜索

提醒和日历一起搜，支持 is:event due:week 这类过滤。

🔒 默认隐私

全部在本地。可选开启端侧 Apple Intelligence 精修——什么都不外传。

🆓 免费开源

MIT 协议。无账号、无追踪、无订阅。

基于 MIT 协议免费开源 · github.com/ekkkkki/QuickAdd
