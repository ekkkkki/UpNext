Nextor 是什么？

Nextor 是一款免费、开源的 macOS 菜单栏 App，专为苹果「提醒事项」和「日历」打造。 在任意界面按 ⇧⌘A，面板打开即是你的日程——逾期、今天、最近几天，按日期分组。像说话一样打字 （中文 / 日本語 / English），Nextor 自动判断是提醒还是日历事件，抽取时间、地点、优先级、清单、标签与重复并保存—— 无需切换 App、无需填表，全部在本地运行。

一打开，今天要做的事就在眼前

⇧⌘A 打开即是你的日程——逾期、今天、最近几天，按日期分组。任意一条都能就地完成或改期，想加新的就直接打字。

自然地打字，剩下的它来搞定

有时间段或时长 → 事件；有地点或会议关键词 → 事件；其余都是提醒。

明天下午3点 开会 30min→ 日历事件，15:00–15:30
周五 9点-10点 团队会议→ 带时间段的事件
客户会议 明天下午3点 建国路88号→ 事件，地址 → 地点 📍
买菜 ~杂货 #周末 !!→ 高优先级提醒，带清单和标签
每周一 上午10点 周会→ 每周重复的提醒

功能

📅 打开即一览

面板打开就显示逾期、今天、最近几天的事，按日期分组——打字之前先看清接下来要做什么。

⌨️ 全局快捷键

Spotlight 式面板浮在任何 App 之上（快捷键可自定义），用完自动把焦点还给你。

🧠 自然语言

时间、时长、区间、重复、提前提醒、优先级、地点——从你打字的方式里直接解析。

📍 智能判别

知道"在某地址开会"是事件，并自动把地址提取成地点。

🔁 强大的重复

每天/每周/每月、多个星期几（每周一三五），以及 共10次、到周五 等结束条件。

🔎 今日与搜索

「今天」一览 + 跨提醒和日历的统一搜索（is:event due:week），可就地改期或完成。

↩ 撤销与批量

加错了？⌘Z。粘贴一个清单，用 ⌥↩ 一次性逐行添加。

🔒 默认隐私

全部在本地。可选开启端侧 Apple Intelligence 精修——什么都不外传。

常见问题

Nextor 是什么？

一款免费、开源的 macOS 菜单栏 App，用于苹果「提醒事项」和「日历」。按 ⇧⌘A 即可看到逾期/今天/最近几天的事，再用自然语言添加提醒或日历事件，自动解析时间、地点、优先级和重复。

怎么添加提醒或事件？

在任何界面按 ⇧⌘A，像说话一样打字，例如 明天下午3点 开会 30min 或 买牛奶 周五。Nextor 自动判断类型并填好细节，保存到苹果提醒事项 / 日历。

支持中文和日文吗？

支持。解析器与界面均完整本地化为中文 / 日本語 / English——星期、时间、时长、重复、地点都按各语言识别。

怎么判断是提醒还是日历事件？

有时间段或时长（9点-10点、3pm 30min）→ 日历事件；具体时间加地点或会议词 → 也是事件并抽取地址为地点；其余为提醒。

免费且开源吗？

是。基于 MIT 协议免费开源，无账号、无追踪、无订阅。

会上传我的数据吗？

不会。全部在本地运行并直接写入苹果提醒事项与日历；可选的端侧 Apple Intelligence 精修也仅在设备上进行。
