Nextor とは？

Nextor は Apple「リマインダー」と「カレンダー」のための、無料・オープンソースの macOS メニューバーアプリです。 どこでも ⇧⌘A を押すと、パネルがそのまま予定一覧（期限切れ・今日・これから）に。思ったまま入力すれば （日本語 / 中文 / English）、リマインダーか予定かを自動判別し、時刻・場所・優先度・リスト・タグ・繰り返しを取り出して 保存します。アプリの切り替えもフォーム入力も不要、すべて端末内で動作します。

特長

📅 開いて一覧

パネルを開くと期限切れ・今日・これからの予定が日付ごとに。入力する前に「次にやること」が見えます。

⌨️ グローバルショートカット

Spotlight 風のパネルがどのアプリの上にも出ます（ショートカットは変更可）。終わると元のアプリにフォーカスを戻します。

🧠 自然言語

時刻・所要時間・範囲・繰り返し・事前アラーム・優先度・場所を、入力した言葉から解析します。

📍 かしこい判別

住所つきの会議は「予定」だと判断し、場所を自動で取り出します。

🔁 強力な繰り返し

毎日／毎週／毎月、複数曜日（毎週月水金）、×10回 や 金曜まで などの終了条件も。

🔎 今日 & 検索

「今日」一覧と、リマインダー・カレンダー横断検索（is:event due:week）。その場で再スケジュール・完了。

↩ 取り消し & 一括

追加ミスは ⌘Z。リストを貼り付け、⌥↩ で各行を一括追加。

🔒 プライバシー重視

すべてローカル。任意でオンデバイスの Apple Intelligence 補正も——データは外に出ません。

よくある質問

Nextor とは？

Apple「リマインダー」と「カレンダー」のための無料・オープンソースの macOS メニューバーアプリ。⇧⌘A で期限切れ・今日・これからの予定を確認し、自然言語で入力するとリマインダーか予定かを自動判別し、時刻・場所・優先度・繰り返しを解析して登録します。

リマインダーや予定はどう追加しますか？

どこでも ⇧⌘A を押して、思ったまま入力します（例：明日15時 会議 30分、牛乳を買う 金曜）。Nextor が種類を判断し詳細を埋めて、Apple リマインダー / カレンダーに保存します。

中国語・日本語に対応していますか？

はい。解析エンジンと UI は 日本語 / 中文 / English に完全対応。曜日・時刻・所要時間・繰り返し・場所をそれぞれの言語で認識します。

リマインダーと予定はどう判別しますか？

時間範囲か所要時間（14時-15時、15時 30分）→ 予定、具体的な時刻＋場所や会議語 → 住所を場所にした予定、それ以外 → リマインダー。

無料・オープンソースですか？

はい。MIT ライセンスの無料・オープンソース。アカウント・追跡・サブスクリプションなし。

データは外部に送信されますか？

いいえ。すべて端末内で動作し、Apple リマインダー / カレンダーに直接書き込みます。任意の Apple Intelligence 補正も端末内で完結します。
